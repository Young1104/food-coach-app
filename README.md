# Food Coach · AI 营养教练

> 给健康饮食新手的 AI 营养教练：用大白话说一句生活，就拿到一份**懂你、会解释「为什么」**的计划；不会做饭也没关系——材料包送到家，**AI 视频通话一步步教你做**；任何时候有问题，**底部全局 AI co-pilot 随叫随到**。

一个移动端健康饮食 App 的产品 + 设计原型，包含完整的 PRD、可交互高保真原型与设计系统规范。

## 项目背景

本项目源自设计作品集中的 **Food Coach**（培养健康饮食习惯 + 让人爱上做饭的 App）。基于其用户研究重新立项并迭代：

- **核心洞察**：健康饮食新手的障碍不是「不知道重要」，而是**「知道却做不到、坚持不下来」**——找信息累、不会规划、不会做饭/嫌麻烦、通用模板不贴合生活就放弃。
- **市场依据**：50.7% 用户愿为健康服务付费，最看重**定制化**与效率，而这正是现有竞品（下厨房 / 小红书 / Keep）的共同空白区。
- **产品方向**：以 **AI** 为中枢，覆盖新手从「想吃健康」到「真的做到」的完整链路：认知+规划 → 执行 → 坚持。

产品历经多轮设计评审打磨（信息架构、视觉系统、文案与数字一致性），本仓库为整理后的成果。

## 产品概览

三层链路 + 五大模块（底部导航，AI 居中为中枢）：

| 层 | 解决 | 体现 |
|---|---|---|
| 认知 + 规划 | 不会规划、看不懂 | AI 生成 + 解释个性化计划（每餐「为什么这样吃」） |
| 执行 | 不会做、嫌麻烦 | 指定菜谱 → 材料包配送 → **AI 视频通话教做菜** |
| 坚持 + 答疑 | 坚持不下来 | 进度可视化、健康科普、**全局 AI co-pilot** |

**模块**：首页（今日计划 + 进度概览 + 科普）· 食谱 · ✦ AI co-pilot · 进度 · 我的。

## 设计系统要点

- **配色**：中性底 + 清新浅绿主卡（深绿字 + 白瓦片 + 进度环）；绿色克制——仅用于主卡、主 CTA、AI、进度环。
- **组件**：按钮 = 圆角矩形 10px；徽章/筛选 = 药丸；线性 SVG 图标；卡片扁平描边、无大阴影。
- **网格**：卡内左对齐 16px，数值列对齐。
- **数字单一数据源**：热量/宏量/进度/目标全部由三餐数据实时计算，跨页一致。

详见 [`design-system.md`](design-system.md)。
